Este projeto é um **guia prático e ilustrado** desenvolvido para auxiliar estudantes da **ETEC** no uso inicial do GitHub.  
A página apresenta instruções passo a passo sobre:

- Como criar uma conta no GitHub utilizando o e‑mail institucional da ETEC.  
- O processo para solicitar o **GitHub Student Developer Pack** e acessar benefícios exclusivos para estudantes.  
- A criação de repositórios e organização de arquivos.  
- A importância do `README.md` e boas práticas de documentação.  
- Recursos úteis como GitHub Pages e links diretos para materiais de apoio.  

Além das instruções, o projeto inclui **ilustrações simuladas**, dicas para professores e um painel lateral com links rápidos para ferramentas essenciais.  
O objetivo é oferecer um material didático acessível, que facilite o aprendizado de versionamento e publicação de projetos no GitHub dentro do contexto educacional da ETEC.
